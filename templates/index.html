API Documentation - JSON HTTP Database

Welcome to the JSON HTTP Database API! Our API provides a secure way to store and manage your credentials or tokens. Below are the available routes and their usage:

/create

Creates a new key and assigns a value to it.

Usage: /create?key=&value=

Example: /create?key=my_key&value=my_value

/get

Gets the value of the provided key.

Usage: /get?key=

Example: /get?key=my_key

/update

Updates the key's value to the new value passed.

Usage: /update?key=&value=

Example: /update?key=my_key&value=new_value

/delete

Deletes a key from the database.

Usage: /delete?key=

Example: /delete?key=my_key
